# Módulo app

Este módulo contém o aplicativo Android em Kotlin/Jetpack Compose.

## O que existe neste módulo

- UI com Compose (telas, navegação e componentes).
- ViewModels com estado reativo usando Flow.
- Camada de dados local com Room (Entity, DAO e Database).
- Injeção de dependências com Hilt.

## Funcionalidades implementadas

- CRUD completo de tarefas.
- Prioridade (Baixa, Média, Alta e Urgente).
- Prazo opcional e faixa de horário opcional.
- Lista com filtros: Ativas, Todas, Concluídas e Atrasadas.
- Dashboard com métricas, progresso e gráfico por prioridade.
- Calendário mensal com tarefas por dia.
